      • java.lang.Object getProperty(java.lang.Object instance, java.lang.String paramName)

        Gets the property from the instance. paramName is a hard-coded 'params' name (e.g. 'username') and the actual property name is looked up from the security config in case it was overridden.

        throws:
        grails.plugin.springsecurity.ui.InvalidValueException if there's a problem getting the property
        Parameters:
        instance - the instance
        paramName - the 'params' name
        Returns:
        the value or null if it's not callable

      • java.lang.String paramNameToPropertyName(java.lang.String paramName, java.lang.String controllerName)

        Lookup the property name based on the 'params' name, e.g. the overridden name for 'username'. The controller name is used since there's no instance to know its class.

        Parameters:
        paramName - the 'params' name
        controllerName - the taglib 'controllerName' property
        Returns:
        the actual property name in the domain class

      • java.lang.Object setProperties(java.util.Map data, java.lang.Object instanceOrClass, org.springframework.transaction.TransactionStatus transactionStatus)

        Sets the properties in the instance (creating it first if instanceOrClass is a Class) from the provided map. The keys are hard-coded 'params' names (e.g. 'username') and the actual property names are looked up from the security config in case they've been overridden.

        throws:
        grails.plugin.springsecurity.ui.InvalidValueException if there's a problem setting a property
        Parameters:
        data - the data
        instanceOrClass - an instance or Class
        the - current TransactionStatus if in a transaction, to pass to the error strategy
        Returns:
        the instance passed or the new instance created from a Class
